# Healthcare Survey and Analytics Platform
## Description
Healthcare organizations, requires to conduct patient surveys to collect patient views that is essential to
providing a patient-centered health service.
Also other organizations may require to conduct surveys for collecting feedback data and statistically or
analytically processing those data for further improvement of their product or services.
This project is aimed at developing an Online Survey System that will help organizations to conduct
surveys and collect valuable response and analyze the response to their benefit.

This Online Survey System application is designed to streamline the survey process, offering features such as customizable time limits, flexible response options, efficient survey distribution via email broadcasts, and comprehensive reporting capabilities. The focus of this project is on developing user-friendly interfaces and implementing robust functionalities to empower organisations in gathering, analysing, and acting upon feedback effectively.

## Objectives
- Efficient Survey Creation and Distribution
- Flexible Survey Settings
- Comprehensive Response Management
- Robust Reporting and Analysis
- Secure Data Handling
- Seamless Compatibility
- User-friendly Design Implementations
- Continuous Enhancement 

## Features
- Seamless Survey Creation
-	Multiple Question Types 
-	In-Depth Data Analysis
-	Efficient Participant Management
-	Question Bank (Private/Public)
-	User-Friendly Interface 
-	Saved Templates

## Tech Stack
- Language: C#, TypeScript
- Framework: Angular, .Net MVC
- Database: SQL Server
- Tools: Visual Studio, Visual Studio Code
